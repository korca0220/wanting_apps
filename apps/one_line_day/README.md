# OneLine Day

OneLine Day는 매일 딱 한 줄만 남기는 개인 기록 앱입니다. DailyPiece처럼 기존 `design_system`을 사용하되, v1은 인증·서버·사진 없이 로컬 저장 기반의 초경량 텍스트 기록에 집중합니다.

## 현재 상태

- 단계: **문서화 우선 기획 / 화면 명세 초안**
- 구현: Android/iOS Flutter 기본 스캐폴드 추가됨
- 화면 명세: [`docs/screens/`](docs/screens/) 참조
